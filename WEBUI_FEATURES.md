# WebUI功能说明

## 新增功能概览

本次更新为Minecraft游戏社科分析台添加了完整的知识库管理和文档导入分析功能，大大提升了系统的易用性和功能性。

## 一、多页面导航

系统现在支持4个主要页面：

### 1. 主页
- 游戏事件描述输入
- AI分析生成
- 分析结果展示
- 理论引用追踪
- 黄金样本保存

### 2. 知识库管理
- 卡片列表查看
- 卡片搜索和筛选
- 添加新卡片
- 编辑现有卡片
- 删除卡片
- 批量导入卡片

### 3. 文档导入分析
- 多文档上传
- 文档内容预览
- 批量分析
- 自动导入到知识库

### 4. 系统设置
- 系统信息查看
- 配置信息展示
- 数据管理
- 黄金样本导出

## 二、知识库管理功能

### 2.1 卡片列表

**功能特点：**
- 支持卡片视图和表格视图切换
- 实时搜索功能
- 多种排序方式（标题、创建时间、更新时间）
- 卡片详情展示

**使用方法：**
1. 进入"知识库管理"页面
2. 选择"卡片列表"标签
3. 使用搜索框查找卡片
4. 切换视图模式查看卡片
5. 点击卡片进行编辑或删除

### 2.2 添加卡片

**功能特点：**
- 表单化输入
- 必填字段验证
- 关键词自动解析
- 即时反馈

**字段说明：**
- **卡片ID**: 唯一标识符（必填）
- **标题**: 理论标题（必填）
- **内容**: 理论内容描述（必填）
- **关键词**: 用于检索和分类（可选）
- **来源**: 理论来源引用（可选）

### 2.3 批量导入

**功能特点：**
- JSON格式支持
- 文件内容预览
- 导入进度显示
- 错误处理

**文件格式：**
```json
[
  {
    "id": "theory_001",
    "title": "理论标题",
    "content": "理论内容",
    "keywords": ["关键词1", "关键词2"],
    "source": "来源"
  }
]
```

**使用方法：**
1. 准备JSON格式的卡片文件
2. 进入"批量导入"标签
3. 上传JSON文件
4. 预览卡片内容
5. 点击"确认导入"

## 三、文档导入分析功能

### 3.1 支持的文档格式

- **TXT**: 纯文本文件
- **MD**: Markdown文件
- **JSON**: JSON格式文件

### 3.2 文档上传

**功能特点：**
- 多文件同时上传
- 文件大小和行数统计
- 内容预览
- 拖拽上传支持

**使用方法：**
1. 进入"文档导入分析"页面
2. 选择"上传文档"标签
3. 拖拽或点击上传文件
4. 查看文件预览

### 3.3 分析选项

**分析模式：**
- **批量分析**: 每个文档单独分析
- **单独分析**: 选择特定文档分析
- **合并分析**: 合并所有文档后分析

**自动导入：**
- 勾选"自动导入到知识库"
- 系统会自动提取文档内容
- 将内容分割为理论卡片
- 导入到知识库

### 3.4 文档处理流程

```
上传文档 → 内容预览 → 选择分析模式 → 开始分析 → 查看结果
                ↓
         自动导入到知识库（可选）
```

## 六、使用建议

### 6.1 知识库管理

1. **定期整理**: 定期检查和更新理论卡片
2. **规范命名**: 使用统一的ID命名规范
3. **完整信息**: 尽量填写完整的卡片信息
4. **关键词优化**: 添加准确的关键词提高检索效果

### 6.2 文档导入

1. **文件准备**: 确保文档格式正确
2. **内容清洗**: 导入前清洗文档内容
3. **分批处理**: 大量文档建议分批导入
4. **验证结果**: 导入后验证卡片内容

### 6.3 性能优化

1. **模型缓存**: 首次加载后模型会缓存
2. **批量操作**: 使用批量导入提高效率
3. **定期清理**: 清理不需要的卡片释放空间

## 七、常见问题

### Q1: 如何批量添加卡片？

**A**: 使用"知识库管理"页面的"批量导入"功能，上传JSON格式的卡片文件。

### Q2: 支持哪些文档格式？

**A**: 目前支持TXT、MD和JSON格式，未来会支持更多格式。

### Q3: 如何查看已上传的文档？

**A**: 进入"文档导入分析"页面的"文档列表"标签（功能开发中）。

### Q4: 卡片搜索不工作怎么办？

**A**: 确保知识库中有卡片，并且搜索关键词与卡片内容相关。

### Q5: 如何导出黄金样本？

**A**: 进入"系统设置"页面，点击"导出黄金样本"按钮。
