## 一、项目概述

不曾社科理论RAG分析系统是一个基于检索增强生成(RAG)架构的社会科学理论分析平台。系统整合了马克思主义理论、社会学、政治学、经济学等多学科知识，能够对游戏内社会现象和现实社会现象进行深入的理论分析。

### 1.1 项目背景

随着大语言模型技术的发展，RAG架构成为知识密集型应用的重要解决方案。本项目旨在构建一个专业的社会科学理论分析工具，帮助用户从理论视角分析社会现象。

### 1.2 项目目标

- 构建完整的社会科学理论知识库
- 实现智能化的理论检索和分析
- 支持多种分析模式和输出格式
- 提供友好的Web交互界面

## 二、系统架构

### 2.1 技术架构

```
+-------------------+     +-------------------+     +-------------------+
|    Streamlit      |     |    ChromaDB       |     |   DeepSeek API    |
|    Frontend       |---->|    Vector DB      |<----|    LLM Service    |
+-------------------+     +-------------------+     +-------------------+
         |                        |                         |
         v                        v                         v
+-------------------+     +-------------------+     +-------------------+
|   document_       |     |    embedding      |     |    llm_api        |
|   processor       |     |    (BGE-large)    |     |                   |
+-------------------+     +-------------------+     +-------------------+
```

### 2.2 核心模块

| 模块 | 功能 | 文件 |
|------|------|------|
| 主应用 | Streamlit Web界面 | app.py |
| 配置管理 | 环境变量和配置 | config.py |
| 知识库 | 向量数据库管理 | knowledge_base.py |
| LLM接口 | DeepSeek API封装 | llm_api.py |
| 向量嵌入 | 本地嵌入模型 | embedding.py |
| 文档处理 | PDF/DOCX解析 | document_processor.py |
| 文档导出 | MD/DOCX导出 | document_exporter.py |
| AI分析 | 自动知识提取 | ai_analyzer.py |

## 三、功能实现

### 3.1 理论分析功能

- **双模式分析**: Minecraft游戏分析 / 普通社科分析
- **智能检索**: 基于向量相似度的理论和历史资料检索
- **流式输出**: 实时显示分析结果
- **引用标注**: 自动标注理论引用

### 3.2 知识库管理

- **理论卡片**: 173张社会科学理论卡片
- **历史资料**: 39条人类文明历史记录
- **黄金样本**: 高质量分析样本保存
- **批量导入**: JSON格式批量导入

### 3.3 文档处理

- 支持格式: PDF, DOCX, TXT
- AI自动分析生成知识卡片
- 可选择自动导入知识库

### 3.4 报告导出

- Markdown格式导出
- Word文档导出
- 包含完整的分析信息和引用

## 四、知识库内容

### 4.1 理论卡片 (173张)

**马克思主义理论 (48张)**
- 共产党宣言、资本论、德意志意识形态
- 哥达纲领批判、政治经济学批判序言

**列宁主义 (8张)**
- 国家与革命、帝国主义论
- 怎么办、唯物主义和经验批判主义

**毛泽东思想 (27张)**
- 实践论、矛盾论、论持久战
- 论人民民主专政、关于正确处理人民内部矛盾的问题

**西方社会学 (90张)**
- 韦伯: 新教伦理、官僚制、统治类型
- 涂尔干: 社会分工论、自杀论
- 布迪厄: 场域理论、惯习、资本形式
- 福柯: 权力/知识、全景敞视
- 哈贝马斯: 公共领域、交往行动

### 4.2 历史资料 (39条)

- 古代文明: 美索不达米亚、古埃及、古希腊、古罗马
- 中国历史: 夏商周、秦汉、唐宋、明清
- 近代史: 文艺复兴、工业革命、法国大革命
- 现代史: 十月革命、世界大战、改革开放

## 五、技术特点

### 5.1 本地化部署

- 嵌入模型本地运行 (BAAI/bge-large-zh-v1.5)
